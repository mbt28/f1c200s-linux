# psplash themes (480x272)

Ten boot-splash designs for the Lctech Pi F1C200s panel, generated by
`generate.py` (Pillow + the FastCarPlay `font.ttf`). The bottom ~40 px of
every theme stays calm so psplash's progress bar doesn't fight the art.

| file | theme |
|---|---|
| 01-play-ring.png | play emblem + motion lines (default) |
| 02-phone-car-link.png | phone <-> car pairing pictogram |
| 03-tux.png | Tux — "powered by Linux" |
| 04-night-road.png | road to the horizon under stars |
| 05-speedometer.png | amber gauge with needle |
| 06-terminal.png | terminal window boot log |
| 07-wordmark.png | minimal tricolour wordmark |
| 08-headlights.png | night drive headlight beams |
| 09-phone-waves.png | phone with radio waves |
| 10-sunrise-dash.png | sunrise over a dashboard arc |

Switch the active splash by pointing the defconfig at a theme:

    BR2_PACKAGE_PSPLASH_IMAGE="$(BR2_EXTERNAL_CARPLAY_PATH)/board/lctech/pi-f1c200s/splash-themes/03-tux.png"

then `scripts/build.sh psplash-dirclean && scripts/build.sh` (the image is
compiled into the psplash binary). Regenerate or tweak the art with:

    FCP_FONT=/path/to/FastCarPlay/src/resource/font.ttf python3 generate.py
